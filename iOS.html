Pyramids

Made using SceneKit

This was my first try with 3D games using Swift. I used the premade 3D geometric figures provided with SceneKit and rendered them on the screen using the renderer function. There after I used the touchesBegan function to roughly check whether the user had touched the intatiated object or not.

Available for iOS 8.0+

Space Jam

Made using SpriteKit

This was my first attempt at iOS game development. I used SpriteKit to create and insantiate the gameobjects on the viewcontroller. Then I used the physics body system to have them all fall towards the spacecraft, and also calculate my collisions. Then for every collisions I displayed the score using UIKit.

Available for iOS 8.0+
